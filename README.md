# Kendall Bias in AR(1)

This repository contains a derivation of the Kendall (1948) bias, i.e., the finite-sample bias of the ordinary least squares (OLS) estimator in an AR(1) model.

## Contents

* `Kendall Bias.ipynb`: Step-by-step derivation using a Taylor expansion of a ratio of random variables.

## Notes

* The derivation follows the standard decomposition of the AR(1) estimator as a ratio and expands around expectations.
* All algebra is carried out explicitly; no reliance on pre-stated results.

## Reference

* Kendall, M. G. (1948). *Note on bias in the estimation of autocorrelation*. Biometrika.
